บทเรียนพื้นฐาน HTML & CSS

เหมาะสำหรับมือใหม่ เรียนรู้โครงสร้างเว็บด้วย HTML และตกแต่งให้สวยงามด้วย CSS แบบเข้าใจง่าย พร้อมตัวอย่างโค้ดและแบบฝึกหัดสั้น ๆ

Beginner HTML CSS

1) HTML คืออะไร?

HTML (HyperText Markup Language) คือ “ภาษาโครงสร้าง” ของหน้าเว็บ ใช้บอกว่าเนื้อหาในหน้าเว็บคืออะไร เช่น หัวข้อ ข้อความ รูปภาพ ลิงก์ รายการ ฯลฯ

จำง่าย: HTML = โครง, CSS = แต่งหน้า

2) โครงสร้างไฟล์ HTML พื้นฐาน

ไฟล์ HTML มักเริ่มต้นด้วยโครงสร้างนี้

<!DOCTYPE html>
<html lang="th">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My First Page</title>
  </head>
  <body>
    <h1>สวัสดี HTML</h1>
    <p>นี่คือย่อหน้า</p>
  </body>
</html>
  • <head> เก็บข้อมูลของหน้า เช่น title, meta, link css
  • <body> คือส่วนที่ผู้ใช้มองเห็นจริง

3) แท็กที่ใช้บ่อย

หัวข้อ & ย่อหน้า

<h1>หัวข้อใหญ่</h1>
<h2>หัวข้อรอง</h2>
<p>ข้อความย่อหน้า</p>

ลิงก์ & รูปภาพ

<a href="https://example.com">ไปที่เว็บ</a>
<img src="photo.jpg" alt="คำอธิบายรูป" />

รายการ (List)

<ul>
  <li>รายการ 1</li>
  <li>รายการ 2</li>
</ul>

กล่อง/โครง (Layout)

<header>ส่วนหัว</header>
<main>เนื้อหาหลัก</main>
<footer>ส่วนท้าย</footer>
<div>กล่องทั่วไป</div>

4) Attributes (คุณสมบัติของแท็ก)

Attribute คือข้อมูลเพิ่มเติมของแท็ก เช่น href ของลิงก์ หรือ src ของรูปภาพ 

<a href="https://example.com" target="_blank">เปิดแท็บใหม่</a>

แนะนำให้ใส่ alt ใน <img> เสมอ เพื่อการเข้าถึง (Accessibility) และ SEO

5) CSS คืออะไร?

CSS (Cascading Style Sheets) คือภาษาสำหรับ “ตกแต่ง” หน้าเว็บ เช่น สี ขนาดฟอนต์ ระยะห่าง การจัดวาง ทำให้ HTML ดูสวยและเป็นธีมเดียวกัน

6) รูปแบบการเขียน CSS

CSS ประกอบด้วย selector และ properties

/* selector { property: value; } */
h1 {
  color: #2563eb;
  font-size: 32px;
}

7) Selector ที่ควรรู้

เลือกตามแท็ก

p { color: #334155; }

เลือกตามคลาส (.)

.card { border: 1px solid #e2e8f0; }

เลือกตาม id (#)

#content { padding: 24px; }

เลือกแบบซ้อน

.card h3 { margin-bottom: 8px; }

8) Box Model (สำคัญมาก)

ทุก element เป็นกล่อง มี 4 ส่วน: content → padding → border → margin

.box {
  padding: 16px;  /* ระยะด้านใน */
  border: 1px solid #cbd5e1;
  margin: 16px;   /* ระยะด้านนอก */
}

9) จัดวางแบบง่ายด้วย Flexbox

.row {
  display: flex;
  gap: 12px;
  align-items: center;
  justify-content: space-between;
}

เคล็ดลับ: ถ้าติดปัญหาจัดกลางแนวนอน/แนวตั้ง ให้เริ่มจาก display: flex ก่อน

10) แบบฝึกหัด (ลองทำ 10 นาที)

  1. เปลี่ยนข้อความปุ่มให้เป็น “เริ่มเรียน”
  2. เพิ่มหัวข้อใหม่ในบทเรียน เช่น “Semantic HTML”
  3. ลองเปลี่ยนสีธีมใน CSS จากน้ำเงินเป็นเขียว โดยแก้ค่าตัวแปรสี
